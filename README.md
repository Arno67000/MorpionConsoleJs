# MorpionConsoleJs
Petit jeu version console du morpion en JavaScript
Little ConsoleGame in JS.

Clone the repo
`npm install`

TO START THE GAME:
`npm start`

TO PLAY :
Each field corresponds to one of your numPad's key 

Align 3 `X` to win and make sure the computer doesn't align 3 `O`.

Good Luck
